## Quick Start

### Prerequisites

- A Linux environment (VM, WSL, or native)
- Bash shell
- Basic understanding of command-line navigation

### Running the Scripts

```bash
# Clone the repository
git clone https://github.com/danialahmed2207/cloud_25_09.git
cd cloud_25_09

# Make scripts executable
chmod +x scripts/*.sh

# Run a script
./scripts/system-info.sh
```
