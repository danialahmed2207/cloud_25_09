# Lab 03: Cloud Basics

> Explore foundational cloud concepts through research and hands-on exercises.

---

## Objectives

- Understand the difference between IaaS, PaaS, and SaaS
- Explore a cloud provider console (AWS, Azure, or GCP)
- Create and manage a simple cloud resource
- Understand billing and cost management basics

## Exercises

### 1. Service Model Identification

Classify the following services:

| Service | Model (IaaS / PaaS / SaaS) |
|---------|----------------------------|
| AWS EC2 | ? |
| Google Docs | ? |
| Heroku | ? |
| Microsoft 365 | ? |
| AWS RDS | ? |
| DigitalOcean Droplet | ? |

*Answers: IaaS, SaaS, PaaS, SaaS, PaaS, IaaS*

### 2. AWS Free Tier Exploration

1. Sign in to the [AWS Management Console](https://aws.amazon.com/console/)
2. Navigate to **EC2 Dashboard**
3. Launch a t2.micro instance (Free Tier eligible)
4. Note the steps: choose AMI, instance type, configure network, add storage, add tags, configure security group, review and launch

### 3. Cost Awareness

- Set up a billing alert in AWS (or Azure / GCP)
- Understand the shared responsibility model
- Review the pricing calculator for a simple VM + storage setup

### 4. Conceptual Questions

- Why might a company choose hybrid cloud over public cloud?
- What is the benefit of auto-scaling?
- How does a load balancer improve availability?
